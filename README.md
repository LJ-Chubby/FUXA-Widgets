# FUXA-Widgets

这是一个基于开源项目 **[FUXA](https://github.com/frangoteam/FUXA)** 创建的自定义SVG小部件库。仓库中收集了作者自制或整理的一些可直接用于FUXA编辑器中的SVG图形部件，旨在帮助用户更快速地构建和美化SCADA/HMI界面。

## 完整小部件创作指南
创作规范如下：
*   参照 [FUXA官方部件库](https://github.com/LJ-Chubby/FUXA-Widgets/blob/main/LICENSE) 的说明文档创作。

## ✨ 部件列表与预览

这个库中包含了一系列实用的工业控制与可视化部件，例如：
*   **控制元素**：按钮、开关、旋钮
*   **状态指示**：指示灯、信号塔、报警器
*   **仪表与显示器**：各种风格的模拟/数字仪表盘、温度计、液位计
*   **工业图标**：泵、阀门、电机、管道、传感器等设备图标
*   **其他装饰**：面板、边框、背景

> **提示**：建议直接访问GitHub仓库查看具体的文件目录和图片，以获得最直观的部件样式预览。

## 📦 如何使用

*   通过Git克隆仓库： `git clone https://github.com/LJ-Chubby/FUXA-Widgets.git`
*   复制到“**FUXA\server\_widgets**”文件夹内。
*   启动你的FUXA项目并进入编辑器。
*   在编辑器左侧小部件内就可以拖使用了。
    
> **提示**：确保将SVG文件放到"**_widgets**"子文件夹内，否则多层文件夹会导致编辑器无法正确加载显示。


## 🔧 定制与修改

所有部件均为标准的SVG格式文件，你可以：
*   在FUXA编辑器内直接调整其**颜色**、**大小**和**旋转角度**。
*   使用任何SVG编辑软件（如Inkscape、Adobe Illustrator等）打开文件进行更深度的自定义编辑，以满足你项目的特定设计需求。
